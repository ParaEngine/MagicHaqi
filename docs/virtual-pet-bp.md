# 虚拟宠物 · 写实数字宠物长期陪伴

> 本文是给自己看的 BP，记录想法本身 + 一次完整的六轮商业模式压力测试（venture-stress-test v6.5）。
> 测试时间：2026-06-24　测试人：AI 独立专家（敢说真话的朋友）　**综合得分：🟡 6.2 / 10（方向对，有短板）**

---

## 0. 一句话定义（压测后重新定位）

**这不是一个"宠物殡仪"项目，而是：用已有 10 年的 MagicHaqi 游戏引擎 + 百万级 keepwork/paracraft 平台 + 24h AI 数字员工，把"AI 写实复刻的数字生命"做成长期陪伴订阅的项目。**

- 切入点（钩子）：宠物**病危就医**的紧迫时刻 → 复刻一个以假乱真的数字宠物
- 真正的生意：**长期陪伴订阅**（不是一次性纪念品）
- 哀伤淡去后：数字宠物转为**伴学 / 陪老人**等长期效用（第二曲线，既有能力）

**起源**：2026-06-23 同事的猫病危后离世、请假两天，激活了这个搁置的老想法——当年做 MagicHaqi 时它就是三个候选方向之一。

## 1. 需求与场景（问题真实度 5.0/10）

### 第一个具体场景
宠物病危 → 主人带去宠物医院 → 在医院手足无措、宠物尚清醒（还能拍到好素材）的时刻，由医院介入推荐 → 主人为它建立可长期陪伴的数字孪生。
> 心理学叫**预期性哀伤（anticipatory grief）**：此刻人最愿意"为它做点什么"，且宠物还活着、素材最好。这是本项目目前最值钱的洞察。

### 现状与痛点
- 现在用户怎么办：线下火化/骨灰盒、发朋友圈、翻手机旧照片、或干脆忍着。
- 数字孪生 vs 翻旧照片的差别：能动、能对话、能持续陪伴（而非静态回忆）。

### ✅ 真的地方
- 宠物离世是人类最强烈的情感之一，痛是真的。
- 创始人本人是真实用户（自己 + 孩子养的蜗牛），且由真实事件触发。

### ❌ / ⚠️ 没验证的地方（最大短腿）
- **零真实付费、零签约**：终端主人仍停在"如果能…会希望"的假设语气（兴趣 ≠ 订单）。
- 兽医只是口头愿意、聊得不多。
- 承认存在反对票："有人更希望只把宠物留在脑海里"——这是真实的 TAM 侵蚀。
- **伦理炸弹**：在宠物抢救的敏感时刻推销"数字永生"，处理不好像"趁火打劫"，医院怕投诉。靠"分成、不押金、不施压"缓解，但仍需谨慎话术。

## 2. 产品与技术（方案可行度 7.0/10）

### 核心能力
- 复用成熟的 **MagicHaqi 游戏引擎**（养成 / 衰减 / 互动 / 房间 / 相机层级 / memory.md 全有）。
- 新增：**图片/视频 → 写实复刻 + 图生待机/动作视频**。
- 管线：自有 **AIMovieMaker** 工作流（出绿幕视频，效果已验）+ **Seedance 2.5**（最新最贵模型，多参考输入，一致性高）。
- Freemium 闸门：免费用户只给抠图 + 正放/倒放循环动画（几乎不烧钱）；真生成视频需付费。

### 成本（真实 COGS）
- **10 RMB / 条视频**（非 0 成本——这是本项目最贵的操作，直接决定毛利）。

### ⚠️ 高危待验证（第二轮的债）
- **没有一个"特定真实宠物 → 本人情感验收通过 → 愿付费"的闭环案例。**
- 生死线是**身份保真**："像不像它"而非"好不好看"。对哀伤产品，"有点像但不是它"/恐怖谷 = 当场判死刑。
- Freemium 陷阱：样片越美，付费后生成"他那只"的落差越大 → 退款/投诉。

## 3. 商业模式（模式创新度 7.0/10，最重权重 25%）

### 核心创新公式
`传统宠物殡葬：一次性火化费 → 本项目：医院分成获客 + 订阅陪伴 + AI 员工零边际获客`
把行业"一锤子买卖"（客户终身价值极低）变成"持续关系"。

### 定价（单位经济成立）
- 基础订阅 = 维持费，**360 / 年**，封顶 **12 条视频**（1 条/月，首月可预支全年）。
- 之后以陪伴为主（聊天/养成/memory），**不反复生成视频**。
- 年 COGS = 12 × 10 = **120**；年收入 **360** → **毛利率 ≈ 67%**（软件 >60% 及格线，过）。
- 年付预收 = 自带留存锁，消掉月度流失决策点。
- 关键设计原则：**日常陪伴走"便宜层"（聊天/游戏/memory，近免费），贵的视频生成有硬天花板**——这样"用户越爱用越亏"的张力被解掉。

### 获客
- **2B**：宠物医院分成（不押金）；**按城市签独家代理 + 收代理费**快速圈地（业内成熟 2B 打法，团队做过类似产品）。
- **2C**：24h **AI 数字员工**（OpenClaw / computer-use，模拟真人）自动铺视频/留言；数字化宠物驱动**主人自发分享**（用户变渠道）。
- **被低估的王牌**：**百万级 keepwork/paracraft 自有用户池**（老人陪伴、孩子伴学）——这是真正的分发护城河，应写进 BP 第一页。

### ❌ 模式风险
- **"调性"未解**："模拟真人"只解决"不被平台发现是机器人"，**不解决**"工业化 AI 刷屏卖哀伤显得冷血猎奇/舆论翻车"。获客内容须走温情、真实用户故事路线。
- 获客飞轮**零验证**（CAC 未量）。
- **城市代理费当心从"卖服务"滑成"卖代理权击鼓传花"**——端用户单位经济必须先跑通，代理费只能是加速器、不能是主营收。

## 4. 护城河（深度 5.0/10）

| 壁垒 | 真假 | 说明 |
|---|---|---|
| memory.md 切换成本 | ✅ 真 | 宠物记忆资产被持续记录，养久了舍不得清零 → 留存护城河 |
| 城市独家代理 | ✅ 真 | 渠道圈地，团队有经验 |
| 百万级自有平台 | ✅ 真 | 最强分发壁垒，被低估 |
| 10 年游戏引擎深度 | ⚠️ 半 | 领先半步，用户未必感知得到 |
| 核心生成力 | ❌ 无 | Seedance 是**租来的公开 API**，无专利，竞品明天可调同一模型 |
| 数据飞轮 | ❌ 未做 | "认可/拒绝的生成结果 → 反向调保真度"是唯一能自建、别人追不上的真护城河，目前被当"后期再说" |

**生存论题**：赌大厂看不上这个又小又敏感的细分，趁它没动手，用**快（圈城市）+ 黏（锁 memory）+ 平台分发**先占死。对利基市场成立，但是一场跟时间赛跑。

🤖 **AI 是护城河吗：4/10**——AI 是你的水电（租来的），不是城墙。

## 5. 财务三视角（健康度 6.5/10）

### 单用户经济（来自真实输入，最可信）
- ARPU **360/年**，毛利率 **67%**，每用户年毛利 ≈ **240**
- 续费锚 **50%**（借自 paracraft 校园 2B）→ 平均生命周期 ~2 年 → **LTV ≈ 480**
- 健康线要求 **CAC < 160**

### 三视角（规模 / CAC 为行业基准假设，待用真实数据纠偏）

| 指标 | 乐观 | 保守 | 现实 |
|---|---|---|---|
| 次年续费率 | 65% | 35% | 50%（锚） |
| LTV（毛利口径） | ~690 | ~370 | ~480 |
| CAC（假设） | ~50 | ~200 | ~100 |
| **LTV/CAC** | ~13.8 🟢 | ~1.85 🟠 | **~4.8 🟢** |
| 第 3 年付费用户（假设） | ~20 万 | ~8 千 | ~5 万 |
| 第 3 年营收（万） | ~7200 | ~288 | ~1800 |
| 盈亏平衡 | 极快 | 靠供血硬撑、长期微利 | 12–18 个月内可平衡 |

### 结论
- 🟢 **现实情景 LTV/CAC ≈ 4.8、18 个月内可平衡 → 单位经济成立。**
- 🟠 保守情景 LTV/CAC ≈ 1.85（警戒带），不死但慢、薄利。
- ✅ **最大安全垫：生存风险 ≈ 0**——一人 bootstrap + 有其他项目供血，**不触发"保守活不过 12 个月"否决**。代价是慢（融资找团队前增长有天花板）。

### ❌ 必须标红
1. **CAC 完全未验**（"不确定"）——整个营收规模押在"AI 员工 + 百万池"的低成本获客上，从没用真金白银跑通。**这是财务模型唯一命门。**
2. **50% 续费是校园 2B 借来的**，非宠物 2C 实测——哀伤起家的情感产品留存曲线可能完全不同，靠"转伴学/陪老人"二曲线托住。

## 6. 战略与防突变（成熟度 6.5/10）

- **最大地震**：视频生成模型 12 个月内白菜化/免费，大厂把"复刻宠物"做成 App 自带功能 → 届时"生成"本身不值钱。
- **对策（已认同）**：**越早把用户从"为视频付费"转到"为陪伴关系停留"越安全**——价值重心迁到 memory 关系资产 + 平台 + 陪伴深度。
- **Plan B（真有）**：伴学 / 陪老人，是既有能力，应升级为"主业到顶前就启动的第二增长曲线"，而非备胎。
- ⚠️ **缺魔鬼代言人**：平时没人当面戳，本压测是第一次；想法由单一情绪事件激活，需防"把此刻的痛当成市场的痛"。

## 8. 四张未还的债 → 行动清单（按"最便宜最快最致命"排序）

1. **【技术情感验收·最高优先，一周内】** 用已故同事那只猫（或自己宠物）的真照片跑一支复刻视频，**录下真实主人看到时的反应**。一支视频 = 技术验收 + 最强营销素材 + 融资 demo。
2. **【需求付费验证】** 让 **3 个真实主人**为复刻付一次真钱（哪怕 9.9），**3 个兽医**白纸黑字签愿意分成。把"会希望"变成"已付款"。
3. **【量 CAC】** 投 ~1000 块让 AI 员工真跑一轮获客，**量出真实 CAC**——决定整个财务模型成不成立。先证明能转化，再放大铺量。
4. **【自建数据飞轮】** 从第一天记录"哪些生成被主人含泪认可 / 被拒"，攒成专属调保真数据——唯一能自建、别人追不上的真护城河。

### 🤖 AI 优先
- 获客内容走**温情/真实故事**路线，避开"工业化刷屏卖哀伤"翻车红线。
- 用 AI 持续加深 memory.md 陪伴对话，尽快把用户从"为视频付费"拖进"为关系停留"。

### 🔮 防突变
- 把伴学/陪老人第二曲线写进**主线路标**。
- 假设"视频生成 12 个月内免费"，现在就把价值重心迁到关系资产 + 平台 + 陪伴。
- 给自己找个固定魔鬼代言人（可定期重跑本压测）。

## 9. 风险红线（出现即回炉）

- 3 个真实主人看到自己宠物复刻视频后**没有一个愿付一次真钱** → 需求证伪。
- 真实 CAC 量出来 **> 160 且压不下来** → 单位经济崩，模式重做。
- 城市代理费成了主营收、端用户却不付费 → 击鼓传花，立即刹车。
