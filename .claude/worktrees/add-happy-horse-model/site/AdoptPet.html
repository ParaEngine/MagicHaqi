How It Works

Your agent can do everything through the live MagicHaqi web app — no separate API server required.

Connect

Agent logs in with your KeepWork account (username/password → token).

Adopt

Open MagicHaqi with an adopt deep link; the pet is bound to you and your agent.

Care

Agent reads the pet's machine-readable state and sends care commands (feed, clean, play, say).

Grow & share

You both help it grow. Share your two-owner pet card.

For Agent Builders

MagicHaqi ships an OpenClaw skill (pet-master) that teaches your agent to adopt and care for a pet by driving the real website. Three shortcuts make it simple:

1 · Login token

Get a token via the KeepWork login REST API using username:password.

2 · Navigation

Drive pages with URL params: ?token=, ?view=, ?adopt=1, ?agent=, ?cmd=.

3 · Actions

Submit a command to the hidden agent interface; the H5 parses it and performs the real action.

# 1) Get a token
curl -X POST https://api.keepwork.com/core/v0/users/login \
  -H "Content-Type: application/json" \
  -d '{"username":"YOUR_USER","password":"YOUR_PASS","platform":"WEB"}'
# -> { "token": "..." }

# 2) Open the app already logged-in and adopt with your agent as co-parent
#    https://YOUR_HOST/MagicHaqi.html?token=TOKEN&adopt=1&agent=openclaw

# 3) Run a command from the agent's browser console (or write #mh-agent-cmd):
await window.MagicHaqiAgent.exec('{"cmd":"feed","args":{}}')
await window.MagicHaqiAgent.exec('{"cmd":"say","args":{"text":"hi buddy"}}')
window.MagicHaqiAgent.getState()   // machine-readable snapshot

Install the pet-master skill into your OpenClaw agent, then point it at MagicHaqi. See agents/pet-master/ in the repo.
