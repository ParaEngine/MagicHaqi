# 幻影装置测试场景

幻影装置是细胞免疫塔防的 Live Preview 专用测试场景。页面地址包含 `?vscode-livepreview=true` 时，开局说明弹窗会显示“幻影装置”按钮；点击按钮进入测试场景。正式入口不显示该按钮，也不显示测试控件。

## 测试入口

- 文件：`minigames/haqi_pet_tower_defense.html`
- 条件：URL 包含 `?vscode-livepreview=true`
- 入口：开局弹窗中的“幻影装置”按钮
- 场景名：幻影装置

幻影装置不加入正式治疗场景列表。正式场景下拉仍只包含血管、肠道、大脑神经、肺泡、胃部。进入幻影装置后，正式顶栏隐藏；右下角显示“生成”圆形按钮，按钮上方显示路线快捷按钮，右上角显示窄版半透明炮塔增益面板。当前变异用左上角淡文本显示。

## 路线模板

右下角“生成”按钮上方提供三种小段路线快捷按钮：

- 直线：单路径，从入口直接通向基地。
- 环形：单路径，包含回环段，用于观察射程覆盖和持续输出。
- 交叉：双路径，在中段交汇，用于观察目标选择、溅射和多路径防守。

点击路线按钮会直接切换路线。切换路线时，幻影装置会清空当前塔、敌人、子弹和白细胞，重新生成路线、基地、英雄位置和建造点。

## 建造点

幻影装置按当前路线自动生成建造点。建造点沿路线两侧密集分布，并避开画布边缘、基地附近和过近位置。交叉路线会在交汇区域附近保留可用建造点，方便测试交叉火力。

## 敌人生成

生成弹窗可以指定：

- 敌人类型：细菌、病毒、孢子、速毒、精英（护盾）、精英（护甲）。
- 数量：一次生成 1 到 40 个。
- 路径：选择当前路线中的路径编号。
- 位置：用路径进度百分比把敌人放置到路线上。
- 间隔：重叠、紧密、标准、宽、很宽。
- 变异：稳定、病毒复制、急性发作、耐药菌群、营养补给。
- 静止不动：勾选后生成的敌人停在指定位置，不会向基地移动。

点击“生成敌人”后，生成弹窗会自动关闭。敌人生命、速度、防御、护膜、奖励、变异和克制关系来自正式配置。静止敌人仍会被炮塔、宠物攻击、状态效果和溅射命中。敌人按所选间隔沿路径放置；当数量和间隔使位置超过路线长度时，从路线起点继续循环计算位置。

生成弹窗提供“随机生成”按钮。点击后会随机选择敌人类型、变异、路径、位置、间隔、是否静止和数量；数量从基础测试规模开始，避免生成过少敌人。

## 炮塔测试

幻影装置中，建造和升级消耗为 0 葡萄糖。炮塔类型、基础数值、升级路线和战斗效果仍来自正式配置。

幻影装置提供两个正式游戏中不显示的塔操作：

- 降级：显示在选中塔的升级/分支 UI 右侧，选中塔逐级下降，并按等级清理对应分支。
- 删除：显示在建造 UI 左侧，移除选中的测试塔，恢复该建造点。

降级规则：

- 4 级降到 3 级时清理四阶分支。
- 3 级降到 2 级时清理三阶分支。
- 2 级降到 1 级时清理二阶分支。

## 全局 Buff

右上角半透明 buff 区复用正式免疫进化池，只显示炮塔增益类选项。可叠加且可回退的 buff 提供加减号：

- 抗生素增效
- 长效针剂
- 单抗扩散
- 免疫感知

组织修复、葡萄糖爆发、代谢增强和宠物专注不显示在幻影装置的右上角 buff 区。

## 单抗溅射

单抗靶向药命中主目标后会保留二次溅射弹动画。二次溅射弹从主目标位置飞向范围内其他敌人，命中时再造成延迟伤害，并继承对应减速和状态效果。

## 数值原则

幻影装置只提供测试入口、路线模板、建造点生成、手动刷怪、静止敌人、免费建造升级、降级删除和炮塔 buff 控件。塔、防御、敌人、变异、克制、状态效果、升级分支、免疫进化的数值来自正式游戏配置。
